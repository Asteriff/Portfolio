Career Field AI - Predicting Jobs

AI-driven tool that predicts suitable job opportunities based on users' skills, experiences, and career aspirations.
Utilizes natural language processing and machine learning algorithms to analyze user profiles and match them
with relevant job listings, facilitating informed career decisions.

Technical

- Visual Studio Code (Required)
- Internet Access

The project is developed in VS Code using the Jupyter Notebook extension. It leverages Pandas and Scikit-learn to build a predictive model, which is seamlessly integrated into a user-friendly website.
