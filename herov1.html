Hero Version 1 - Diagnosis Prediction

Type in your symptoms and Hero will try to predict an area of diagnosis based on twelve categories.
Paired with an Android application for future development.
Displays a template to show how a user might interact.

Technical

- Android Studio (Required)
- Visual Studio Code (Required)

The project is built using Java and XML, with Gradle Scripts for efficient development. It utilizes Firebase for cloud storage, securely managing user login information and authentication codes.
